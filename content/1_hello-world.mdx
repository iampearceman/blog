# Welcome to my blog!

This is my first post written in MDX. I'm excited to share my thoughts and experiences with you.

## Some features:

- **Markdown support** - Write content in familiar Markdown syntax
- **Code highlighting** - Share code snippets with syntax highlighting
- **Custom components** - Embed React components directly in your posts
- **Interactive elements** - Add dynamic content
- **Rich formatting** - Tables, quotes, and more

### Code Examples

Here's a React component:

```tsx
const Counter = () => {
  const [count, setCount] = useState(0);
  
  return (
    <div className="p-4 border rounded">
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
    </div>
  );
};
```
